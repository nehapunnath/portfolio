PROJECTS

  1. A SMART CHAIR SITTING POSTURE RECOGNITION SYSTEM USING FLEX SENSORS FOR EMPLOYEES

    Sitting is the most common status of modern human beings. Some sitting postures may bring health issues. To prevent the harm from bad sitting postures, a local sitting posture recognition system is desired with low power consumption and low computing overhead. The system should also provide good user experience with accuracy and privacy. This project reports a novel posture recognition system on an office chair that can categorize seven different health-related sitting postures.The system uses six flex sensors, an Analog to Digital Converter (ADC) board.

  2. To-Do List Web Application

    This project is a dynamic to-do list web application built using the Django framework. The application allows users to register, login, and manage their tasks efficiently. Users can perform various operations such as adding, editing, deleting tasks, and marking tasks as completed. The project utilizes Django's built-in authentication system for user registration and login, ensuring secure access to user-specific data. Tasks are stored in a database and associated with the logged-in user, providing a personalized task management experience. The application features responsive design principles, ensuring compatibility across different devices and screen sizes
